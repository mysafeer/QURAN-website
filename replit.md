# Quran Website

## Overview
A web application for reading and listening to the Noble Quran with Urdu translation.

## Project Structure
- `index.html` - Single-page application containing all HTML, CSS, and JavaScript

## Technology Stack
- Static HTML/CSS/JavaScript
- Uses external fonts (Google Fonts: Amiri, Inter)
- Uses Font Awesome icons

## Development
- Run with Python's built-in HTTP server on port 5000
- Command: `python -m http.server 5000 --bind 0.0.0.0`

## Deployment
- Configured as a static site deployment
- Public directory: `.` (root)
